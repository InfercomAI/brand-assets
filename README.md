## Usage Guidelines

### File Formats
- **SVG** (vector) - Preferred for all digital uses, infinitely scalable
- **PNG** (raster) - For compatibility and high-resolution displays
- **ICO** (favicon) - Legacy browser support

### Logo Guidelines

**Minimum Sizes**
- Logo: Minimum width 120px for digital, 25mm for print
- Favicon: Use provided sizes (16x16, 32x32, etc.)

**Clear Space**
Maintain clear space around logo equal to the height of the "I" in Infercom.

**Do Not**
- Change logo colors
- Distort, rotate, or modify proportions
- Add effects (shadows, gradients, outlines)
- Place on busy backgrounds that reduce legibility

**Background Usage**
- Use **white logo** on dark backgrounds
- Use **black logo** on light backgrounds
- Use **logo on green** when brand color background is needed
